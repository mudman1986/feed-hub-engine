Preview Sites

Preview deployments are built from feature branches so you can review changes without touching the live site. Each branch gets its own isolated URL under preview/<preview-slug>/, where the preview slug is the branch name converted to lowercase, with / replaced by -- and other non-alphanumeric characters replaced by -.
